Actividad III

Josep Chalá

Esta es mi presentación para la Actividad 3 del curso Introducción a la Ingeniería de Sistemas. En esta actividad, nos enfocamos en la migración de datos a BigQuery y en la creación de consultas SQL para obtener información específica sobre la tienda virtual. Además, desarrollamos un dashboard en Looker Studio para presentar visualmente los resultados obtenidos.El objetivo principal de esta actividad es utilizar herramientas y técnicas de análisis de datos para responder preguntas relevantes sobre la tienda virtual y presentar los resultados de manera clara y atractiva.

Consultas 


1. ¿Cuál es el producto más vendido en una ubicación geográfica específica?
2. ¿Cuántos clientes han comprado productos de un rango de precios específico?
3. ¿Cuáles son los clientes más frecuentes en realizar compras y cuánto han gastado en total?
4. ¿Cuál es el producto más vendido en cada ubicación geográfica?
5. ¿Cuáles son los clientes que han comprado todos los productos disponibles?

1. SELECT p.producto AS nombre_producto, d.departamento, COUNT(*) AS total_ventas
FROM `actividadiii.datos.clientes` d
JOIN `actividadiii.datos.compras` co ON d.Numero = co.cliente
JOIN `actividadiii.datos.productos` p ON co.producto = p.Codigo
WHERE d.Departamento = 'Santander'
GROUP BY p.Producto, d.Departamento
ORDER BY total_ventas DESC
LIMIT 1;

2. SELECT COUNT(DISTINCT c.cliente) AS total_clientes
FROM `actividadiii.datos.compras` c
JOIN `actividadiii.datos.productos`  p ON c.producto = p.codigo
WHERE p.precio >= 5000 AND p.precio <= 20000;

3. SELECT d.numero, d.Nombre_1,COUNT(*) AS frecuencia_compra, SUM(c.veces_compradas * p.precio) AS total_gastado
FROM `actividadiii.datos.clientes` d
JOIN (
    SELECT Cliente, Producto, COUNT(*) as veces_compradas
    FROM `actividadiii.datos.compras`
    GROUP BY Cliente, Producto
) c ON d.Numero = c.cliente
JOIN `actividadiii.datos.productos` p ON c.producto = p.Codigo
GROUP BY d.Numero, d.Nombre_1
ORDER BY frecuencia_compra DESC, total_gastado DESC;

4. WITH ventas_por_producto_y_departamento AS (
  SELECT
    p.producto AS nombre_producto,
    d.departamento,
    COUNT(*) AS total_ventas,
    ROW_NUMBER() OVER (PARTITION BY d.departamento ORDER BY COUNT(*) DESC) AS ranking
  FROM
    `actividadiii.datos.clientes` d
    JOIN `actividadiii.datos.compras` co ON d.Numero = co.cliente
    JOIN `actividadiii.datos.productos` p ON co.producto = p.Codigo
  GROUP BY p.Producto, d.Departamento
)
SELECT nombre_producto, departamento, total_ventas
FROM ventas_por_producto_y_departamento
WHERE ranking = 1
ORDER BY departamento

5. WITH clientes_productos AS (
  SELECT cli.Apellidos, cli.Nombre_1, cli.Nombre_2, com.producto
  FROM `actividadiii.datos.clientes` cli  
  INNER JOIN `actividadiii.datos.compras` com
    ON cli.numero = com.cliente
),
productos_disponibles AS (
  SELECT DISTINCT producto
  FROM `actividadiii.datos.compras`
)

SELECT c.Apellidos, c.Nombre_1, c.Nombre_2
FROM clientes_productos c
LEFT JOIN (
  SELECT cp.Apellidos, cp.Nombre_1, cp.Nombre_2, p.producto
  FROM clientes_productos cp
  CROSS JOIN productos_disponibles p
) AS t
ON c.Apellidos = t.Apellidos
  AND c.Nombre_1 = t.Nombre_1
  AND c.Nombre_2 = t.Nombre_2
  AND c.producto = t.producto
WHERE t.Apellidos IS NULL
GROUP BY c.Apellidos, c.Nombre_1, c.Nombre_2
